# Chrome T-Rex Game

![T-rex](https://raw.githubusercontent.com/knickers/chrome-t-rex-game/master/t-rex.gif)

### 1. Right-click on any webpage and choose *Inspect Element*

![Right-Click](https://raw.githubusercontent.com/knickers/chrome-t-rex-game/master/right-click.gif)

### 2. Click *Offline* to simulate disconnecting from the internet

![Offline](https://raw.githubusercontent.com/knickers/chrome-t-rex-game/master/offline.gif)

### 3. Refresh the page

![Refresh](https://raw.githubusercontent.com/knickers/chrome-t-rex-game/master/refresh.gif)

### 4. Press the spacebar to start the game

![Start](https://raw.githubusercontent.com/knickers/chrome-t-rex-game/master/start.gif)
